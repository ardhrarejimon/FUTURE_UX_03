# FUTURE_UX_03 – ClientHub CRM Dashboard UI

## Overview

ClientHub is a modern CRM (Customer Relationship Management) dashboard designed to help businesses manage clients, track leads, monitor appointments, and organize daily follow-up tasks. This project was created in Figma as part of the Future Interns UI/UX Internship – Task 3.

---

## Features

- Client Management Dashboard
- Lead Pipeline Overview
- Client Profile & Engagement Status
- Client List Management
- Task Tracker & Follow-up Reminders
- Responsive Dashboard Layout
- Modern and Clean UI Design
